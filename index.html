Real-Time Trading Simulator using live stock data from a Yahoo API

Java, Spring Boot, Unit-Testing, H2 Database, Swing GUI

Real-Time Trading simulator using live stock data from a yahoo API, with User data and purchase history stored in a one-to-many CRUD Repository. This project used a swing GUI as the front end as we have not been taught any front end for our bootcamp. This was the final capstone project completed as a team during the last week of our 11 weeks JAVA bootcamp.

My role in the team:

I was the backend engineer for this project using spring boot to create all the entities and classes that connect to the H2 database.
