Real-Time Trading Simulator using live stock data from a Yahoo API

Java, Spring Boot, Unit-Testing, H2 Database, Swing GUI

Real-Time Trading simulator using live stock data from a yahoo API, with User data and purchase history stored in a one-to-many CRUD Repository. This project used a swing GUI as the front end as we have not been taught any front end for our bootcamp. This was the final capstone project completed as a team during the last week of our 11 weeks JAVA bootcamp.

My role in the team:

I was the backend engineer for this project using spring boot to create all the entities and classes that connect to the H2 database.

About me

I transitioned into a technology role because I saw the breadth of opportunities available and I wanted to be part of the future disruption caused by the tech sector.

I wasn't always a developer. In the previous chapter of my life, I worked in financial services for a large UK bank. Then after Graduating with a First Class Honours in Accounting with Finance from The University of Glasgow, I worked as a finance assistant at one of the Russell Group Universities.

Not long after, I quit my job as an accountant and joined Barclays as an Aspiring Developer. I have gone through a 16 week Java boot camp which taught me the skills necessary to become a backend Java developer.

I hope to grow my expertise in the tech industry and will continue to learn new skills, for example teaching myself front-end technologies like building this website you're currently viewing. I have much to learn!

I would love to work abroad in Japan someday and immerse myself in a different culture.

My interest includes travelling, fitness, Japanese, programming, gaming and books.
